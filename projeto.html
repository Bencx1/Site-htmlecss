Projetos Sociais

Em todo final do ano, utilizamos uma porcentagem do dinheiro adquirido com compras dos tijolos, para algumas ações sociais, uma delas são o projeto construindo sonhos, onde utilizamos parte da verba dos tijolos ecológicos para construir casas, para familias carentes, onde são entregues a familia carentes. Outro projeto é a Agua é vida, onde em todo final de ano, são distribuidos caminhões pipa para locais onde não possuem agua potável, onde são distribuidas entre comunidades carentes em todo pais.
